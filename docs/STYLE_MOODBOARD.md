# Kingdom Duel Style Moodboard

Date: 2026-06-27

## Style Name

Primary working name: **AeroCandy 2007**

Alternates:

- Glass Future Match-3
- Frutiger Candy Aero
- Skyglass Duel

## Design Read

Kingdom Duel should feel like a polished early-2000s advertisement for a clean, bright, playable future: glossy, watery, optimistic, friendly, a little naive, and instantly readable as a game.

This is not generic glassmorphism. It is a game-specific adaptation of Frutiger Aero / Windows Aero / Web 2.0 gloss:

- glassy skeuomorphic surfaces,
- translucent aqua panels,
- rounded Vista-like UI,
- white specular highlights,
- blue sky and clean green accents,
- bubbles, water drops, soft flares,
- bright candy-like icons,
- no dark fantasy sludge.

## Core Formula

70% clarity + 20% glass gloss + 10% nostalgic excess.

For match-3:

- Tiles get the highest gloss.
- Board frame gets moderate glass.
- Background stays calm and low-noise.
- VFX should be bright and short.

## Palette

Base:

- sky blue `#7ddfff`
- aqua `#25d7f2`
- glass white `#f8ffff`
- mint green `#7bf28d`
- clean silver `#c7dbe8`

Tile accents:

- sun yellow `#ffe25c`
- candy orange `#ff9d3d`
- bubble pink `#ff74c8`
- soft violet `#9b83ff`
- ocean blue `#247cff`
- lime gel `#9bff4f`

Shadows:

- blue-grey only, not dirty black.

## Tile Rules

Each tile must work at 64x64 and 128x128.

Every tile needs:

- distinct silhouette,
- distinct color,
- large top-left white highlight,
- subtle inner glow,
- transparent/glass depth,
- small contact shadow,
- no tiny surface detail,
- no text.

Approved shapes:

- water drop,
- glass orb,
- rounded square jewel,
- capsule,
- star bubble,
- leaf-in-glass,
- sun drop,
- moon pearl,
- aqua crystal,
- lime gel cube.

## UI Rules

Panels:

- translucent aqua glass,
- blurred background,
- white top rim,
- soft inner shadow,
- rounded corners,
- one clean silver/aqua border.

Buttons:

- capsule or rounded rectangle,
- white glossy top,
- aqua depth,
- soft blue shadow,
- pressed state must visibly sink.

Board:

- bright but calmer than tiles,
- no noisy background behind cells,
- cell gaps must remain readable,
- active turn frame should be obvious.

## Avoid

- flat minimalism,
- dark cyberpunk,
- brown medieval palette,
- dirty grunge,
- realistic photo textures,
- excessive reflections,
- harsh black outlines,
- unreadable details,
- noisy backgrounds,
- text/logos/watermarks in tile art.
